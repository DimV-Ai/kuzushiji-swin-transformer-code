# Kuzushiji Swin Transformer Code

This repository contains the training, evaluation, and visualization scripts used for the kuzushiji character classification experiments.

## Contents

- training scripts for CODH, Ogihan, and combined CODH+Ogihan settings
- native evaluation scripts
- plotting script for comparison figures

Model weights and evaluation outputs are available on Hugging Face:
